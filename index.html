The Silent Timer

SilentTimer.com was the site where I first learned anything about programming. My business partner at the time started the site as he had a background in web design and programming. After getting frustrated having to ask him to make the changes I needed, I started learning how to do it myself. After a time, he no longer needed to work on it as I was solely responsible for additions and changes. This is where I first learned to code in PHP, MySQL, and HTML.

I built much of the custom CRM backend where we tracked orders, customers, and other reporting functions. It was so much fun that I worked on the site whenever I could, and eventually when it was performing well, I still created functionality that we really "needed" just so I could continue programming. :-) The company is no longer around, and neither is the website itself, but the I have the code and posted it to GitHub.
